# Digital Signal Processing Notes

This repository contains a collection of notes, code examples, and projects related to Digital Signal Processing (DSP).

## Contents

The repository is organized into the following main sections:

*   **Core DSP Concepts:** Folders like `ComplexDFT`, `Convolution`, `DFT`, and `IDFT` contain implementations and examples of fundamental DSP algorithms.
*   **Image Processing:** The `image_processing` directory holds various projects and examples related to 2D signal processing, specifically for images. Topics include filtering, edge detection, and transformations.
*   **DSP with C++:** The `dsp_with_cpp` folder contains C++ implementations of DSP concepts.
*   **Signal Analysis:** Directories such as `Signal_Mean`, `SignalVariance`, and `Signal_Standard_deviation` provide examples of basic signal analysis techniques.
*   **Sample Data:** The repository also includes various sample signals and data used in the projects, located in folders like `_20hz_sig`, `_ecg_320`, and `waveforms`.

Feel free to explore the different folders to find code and examples related to your interests in DSP.
